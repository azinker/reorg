Mark Return Received
POST /post-order/v2/return/{returnId}/mark_as_received

This method can be used on behalf of a seller to mark a return item as received.


This method is not supported in the Sandbox environment.

Input

Resource URI (production)
POST https://api.ebay.com/post-order/v2/return/{returnId}/mark_as_received
URI parameters
Parameter	Type	Required?	Meaning
returnId	string	Required	The unique eBay-assigned ID of the return. The returnId value is required as part of the call URI to identify the return request for which an item is being marked as 'received'.
