# Sync Page Design

> Overrides MASTER.md for the Sync page.

## Purpose

Pull-only sync controls. Fetch latest data from all connected marketplaces.
Clear status display per integration.

## Layout

- "Sync All" button at top (prominent)
- Per-store sync cards in a grid (2x2 or responsive)
- Each card shows: store logo/acronym, last sync time, status, item counts, sync button

## Visual Treatment

- Store cards with marketplace branding (logo or acronym badge)
- Status indicators: Idle (gray), Syncing (blue animated spinner), Completed (green check), Failed (red x)
- Progress indicator during sync (items processed / estimated total)
- Last sync time in relative format with absolute on hover
- "Pull-only" label clearly visible to reinforce safety
